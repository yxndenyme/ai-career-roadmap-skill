数据处理

本项目是一套在本地或使用者选择的平台内运行的 Agent Skill。发布者不运营独立服务器，不主动收集、上传、出售或长期保存简历和个人画像。

使用者主动提供的资料只用于当前职业分析任务。Skill 会要求排除不必要的电话、邮箱、地址、证件号、推荐人、精确薪酬和雇主机密。

平台责任

实际数据处理取决于运行平台。ChatGPT、Codex、Claude、扣子或 OpenClaw 可能依据自己的隐私政策处理对话、文件、日志、联网搜索和工具调用。

联网研究

Skill 可能访问公开互联网核验公司、职位、政策和学习资料。除非使用者明确要求并在操作前确认，不得把简历发送给招聘网站、公司或其他第三方。

Privacy Policy (English)

This project is an agent skill that runs locally or inside the user's selected AI platform. The publisher does not operate an independent backend and does not intentionally collect, upload, sell, or retain resumes or personal profiles.

User-provided information is used only for the current career analysis. Actual processing depends on the selected platform, including its handling of conversations, files, logs, web searches, and tools. Users should remove unnecessary contact details, identity numbers, addresses, references, and confidential employer information before uploading.
