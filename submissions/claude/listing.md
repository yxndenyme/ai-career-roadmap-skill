## Description

A Chinese-first career planning skill that analyzes a resume or guided intake, researches current AI and AI-adjacent opportunities, recommends realistic role clusters, and creates a personalized learning roadmap covering common, role-specific, and industry-specific capabilities.

## 提交步骤

1. 在仓库根目录运行 `claude plugin validate`。
2. 使用 `claude --plugin-dir .` 测试技能发现和调用。
3. 在 Claude Console 插件提交页面提交 GitHub 仓库。
4. 使用 OpenAI 测试用例中的5个正向和3个负向场景进行交叉测试。
5. 审核通过后确认 `claude-community` 目录已经同步。
