# OpenAI 提交检查清单

- [ ] OpenAI Platform 发布组织已完成个人或企业身份验证。
- [ ] 提交者拥有 Apps Management Write 权限。
- [ ] GitHub Pages 官网、隐私、条款和支持页面可以公开访问。
- [ ] Logo 和商店介绍已经确认。
- [ ] 上传最终插件 ZIP，并选择 Skills only。
- [ ] 填入3个 Starter Prompts。
- [ ] 填入5个正向测试和3个负向测试。
- [ ] 只选择平台支持且能够提供服务的地区。
- [ ] 完成政策声明并检查无个人简历、Token、密钥和本地路径。
- [ ] 提交审核前由发布者最终确认。
